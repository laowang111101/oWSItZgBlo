# 校园招聘系统 python511

## 项目概述

校园招聘系统基于Python的Django框架开发，旨在为高校毕业生、企业和高校管理员提供便捷的招聘与求职服务。系统包括用户端、企业端和管理端，具备简历管理、职位搜索、简历筛选等功能。

## 技术栈

- 开发语言：Python
- Web框架：Django
- 数据库：MySQL
- 开发工具：PyCharm等Python开发工具

## 功能模块

### 用户端

- 登录注册
- 填写/修改简历
- 修改头像和密码
- 查询职位
- 投递简历
- 面试邀请
- 招聘大数据可视化
- 查看企业信息

### 企业端

- 登录注册
- 填写企业信息
- 职位设置与修改
- 职位条件搜索
- 筛选简历
- 面试邀请与结果评价

### 管理端

- 审批企业和用户信息
- 修改信息和操作

## 系统角色

- 求职高校毕业生
- 企业招聘负责人
- 高校管理员

## 运行环境

- 操作系统：兼容Windows、Linux和MacOS
- Python版本：3.x
- 数据库：MySQL

## 部署步骤

1. 环境准备：安装Python、MySQL等必需软件
2. 数据库配置：根据项目需求创建数据库和用户
3. 项目部署：使用PyCharm等工具导入项目，配置数据库连接
4. 运行项目：启动Django服务，访问前端页面
